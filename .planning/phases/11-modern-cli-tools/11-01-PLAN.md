**modules/install_lazygit.sh:**
- Metadata header: `@name: Install lazygit`, `@params:`, `@platforms: linux, darwin`, `@version: 1.0.0`, `@deps: curl,wget`, `@timeout: 600`
- `set -eu` after header

- Copy FLU_PKG_MGR auto-detect fallback block verbatim
- Idempotent guard: `command -v lazygit` → print `lazygit already installed: $(lazygit --version)`, exit 0
- Install logic:
  - Detect arch with `uname -m`: map `x86_64` → `x86_64`, `aarch64` → `arm64`, `armv7l` → `armv7`

  - `_maybe_sudo chmod +x /usr/local/bin/lazygit`
  - Clean up: `rm -f /tmp/lazygit.tar.gz`

  <action>
Create two module scripts. The install follows a curl-pipe pattern similar to install_tailscale.sh. The remove follows the RC-cleanup pattern from remove_rust.sh.

**modules/install_starship.sh:**
- Metadata header: `@name: Install Starship`, `@params:`, `@platforms: linux, darwin`, `@version: 1.0.0`, `@deps: curl,wget`, `@timeout: 600`
- `set -eu` after header
- Copy `_maybe_sudo()` helper + FLU_PKG_MGR fallback verbatim
- Idempotent guard: `command -v starship` → print `Starship already installed: $(starship --version)`, exit 0
- Install logic:
  - **darwin:** If brew available → `brew install starship`, else error
  - **linux (or unset FLU_OS):** Use official installer:
    ```
    curl -sS https://starship.rs/install.sh | sh -s -- -y
    ```
    Or wget equivalent if curl unavailable. The official install.sh installs to `/usr/local/bin/starship` by default.
  - **Other OS:** Error with "Visit https://starship.rs for manual install"
- After successful install, add shell integration:
  - Detect current shell from `$SHELL` (default to bash)
  - For each RC file that exists (`$HOME/.bashrc`, `$HOME/.zshrc`):
    - Check if starship init line already present (`grep -q 'starship init' "$rcfile"`)
    - If not present, append: `printf '\neval "$(starship init bash)"\n' >> "$rcfile"` (use `bash` or `zsh` matching the shell)
  - For fish (`$HOME/.config/fish/config.fish`): append `starship init fish | source` if fish config exists
  - Print message: `printf 'Starship installed. Restart your shell or source your RC file to activate.\n'`
- Success message: `printf 'Starship installed successfully\n'`

**modules/remove_starship.sh:**
- Metadata header: `@name: Remove Starship`, `@params:`, `@platforms: linux, darwin`, `@version: 1.0.0`, `@deps:`, `@timeout: 300`
- `set -eu` after header
- Copy `_maybe_sudo()` helper + FLU_PKG_MGR fallback
- Guard: `! command -v starship` → "Starship is not installed", exit 0
- Remove:
  - **darwin:** `brew uninstall starship 2>/dev/null || true`
  - **linux:** `_maybe_sudo rm -f /usr/local/bin/starship`
- Clean RC files — for each of `$HOME/.bashrc`, `$HOME/.zshrc`, `$HOME/.config/fish/config.fish`:
  - `sed -i.bak '/starship init/d' "$rcfile" 2>/dev/null || true`
  - `sed -i.bak '/# Starship/d' "$rcfile" 2>/dev/null || true`  (catch any comment lines)
  - `rm -f "${rcfile}.bak" 2>/dev/null || true`
- Clean starship config: `rm -rf "$HOME/.config/starship" 2>/dev/null || true`
- Success message: `printf 'Starship removed successfully. Shell integration lines cleaned from RC files.\n'`

Both scripts must be executable.
