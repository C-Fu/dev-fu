Implement three functions:

**1. `_flu_spinner_render()` — internal frame renderer (called by background loop)**
- Reads global `_flu_spinner_frame` (integer counter, incremented by background loop)
- Uses modulus to select a character from a spinner character sequence
- Spinner sequence: ASCII-safe `|/-\|/-\` (works without UTF-8). Optionally detect UTF-8 locale (same pattern as `_tui_detect_box_chars` at lines 37-58) and use braille `⠋⠙⠹⠸⠼⠴⠦⠧⠇⠏` when UTF-8 is available.
- Renders the selected character using `_tui_printf_at` at a fixed position — bottom of screen, centered. Determine row from `tput lines 2>/dev/null || echo 24`, column from `tput cols 2>/dev/null || echo 80`.
- If TUI is not active (`_tui_use_tui=false`), outputs a simple text line: `"Working... {char}"` to stdout (no ANSI codes) — so numbered fallback mode shows progress too.
- Prefix with `"  "` (two spaces) and a label like `"${TUI_CYAN}Loading${TUI_RESET} "` before the spinner character.
- After rendering, increment `_flu_spinner_frame` by 1.

**2. `flu_spinner_start()` — start the spinner background process**
- Guard: if `${_flu_spinner_pid:-}` is non-empty, return immediately (spinner already running).
- Set `_flu_spinner_frame=0`.
- Launch a background subshell: `(while :; do _flu_spinner_render; sleep 0.1; done) &`
- Store the subshell PID in `_flu_spinner_pid=$!`.
- If not in TUI mode, the background process writes to stdout; in TUI mode, it uses `_tui_printf_at` without disturbing the existing screen layout.

**3. `flu_spinner_stop()` — stop the spinner and clean up**
- If `${_flu_spinner_pid:-}` is empty, return immediately (no spinner running).
- Kill the background process: `kill "$_flu_spinner_pid" 2>/dev/null` then `wait "$_flu_spinner_pid" 2>/dev/null`.
- Clear `_flu_spinner_pid=''`.
- Clear the spinner screen area using `_tui_printf_at` to overwrite the spinner line with spaces, or in non-TUI mode, output a newline to move past the spinner line.

**Pattern requirements (per CONCERNS.md and CONVENTIONS.md):**
- Use `$(printf '\033')` for ESC — NOT `$'\033'` (bashism) or `\033` or `\x1b` (sed portability).
- Use `_tui_printf_at` for positioned output (already defined in tui.sh Section 6).
- Use `_tui_clear_line` to clear the spinner line on stop.
- Follow existing tui.sh section header format exactly:
  ```
  # ---------------------------------------------------------------------------
  # Section 16: Async spinner widget
  # ---------------------------------------------------------------------------
  ```
- Prefix all internal variables with `_flu_spinner_` to avoid collisions with existing `_tui_*` variables.
- Clean up all local variables with `unset` at function exit (matching existing tui.sh pattern).
- Add `# shellcheck disable=` pragmas as needed (SC2034 for constants, SC2154 for eval-assigned).

**UNICODE detection pattern (copy from _tui_detect_box_chars lines 37-58):**
```sh
_flu_spinner_detect_utf8() {
  _locale="${LANG:-}${LC_ALL:-}${LC_CTYPE:-}"
  case "$_locale" in
    *UTF-8*|*utf-8*|*utf8*|*UTF8*) _flu_spinner_utf8=true ;;
    *) _flu_spinner_utf8=false ;;
  esac
  unset _locale
}
```

Call `_flu_spinner_detect_utf8` at module scope (like `_tui_detect_box_chars` is called at line 60).

**Spinner character selection:**
- If `_flu_spinner_utf8=true`: use `⠋⠙⠹⠸⠼⠴⠦⠧⠇⠏` (10 braille chars)
- If `_flu_spinner_utf8=false`: use `|/-\` (4 ASCII chars)
- Frame index modulo char count selects the current character.
