```rust
/// Single-select list widget.
/// Returns Ok(Some(index)) on selection, Ok(None) on cancel (Esc/q).

pub fn select(
    terminal: &mut Terminal<CrosstermBackend<Stdout>>,
    theme: &Theme,
    title: &str,
    subtitle: &str,
    items: &[&str],
) -> anyhow::Result<Option<usize>>
```

```rust
struct SelectState {
    cursor: usize,      // current highlighted item (0-based)

    go_digits: String,  // number-key jump accumulator (per D-09)
    error_msg: Option<String>,
    page_size: usize,   // visible items count (computed from terminal height)
}
```

**Key handling (per D-07 — widget-local):**
- `Key::Up` or `Key::Char('k')` → cursor up (wrap to bottom)
- `Key::Down` or `Key::Char('j')` → cursor down (wrap to top)
- `Key::Home` or `Key::Char('g')` → jump to first item
- `Key::End` or `Key::Char('G')` → jump to last item
- `Key::PgUp` → scroll up one page
- `Key::PgDn` → scroll down one page
- `Key::Number(c)` → accumulate digits for go-to (per D-09). On non-number key, resolve accumulated digits to item index.
- `Key::Enter` → return selected index
- `Key::Esc` or `Key::Char('q')` → return None (cancel)
- `Key::Help` → toggle show_help (per D-10)

**Rendering using ratatui built-in widgets (per D-03):**
- `Block` with borders for the outer box, title centered, border color from theme
- `List` with `ListState` for scrollable item list (per D-11 — use ratatui default scrolling)
- `Paragraph` for subtitle (dimmed), footer (compact/extended based on show_help), error message

**Footer text:**
- Compact: `"↑↓/jk:navigate  Enter:select  Esc:cancel  ?:help"`
- Extended (when show_help=true): adds `"PgUp/PgDn:page  g/G:home/end  1-9:go-to  q:quit"`

**Scroll behavior (per D-11):** Use ratatui's ListState::select() + offset calculation. When cursor moves outside visible area, adjust scroll to keep cursor visible. page_size = terminal_height - 5 (title + subtitle + footer + borders).
